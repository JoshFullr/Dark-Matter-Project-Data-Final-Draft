# Dark-Matter-Project-Data-Final-Draft
In this repository are the plots I created to compare dark matter fraction of galaxies with various datatypes as well as additional tables for analysis.

Compared to my first draft of data in a seperate repository, I switched over to TOPCAT instead of Google Sheets to create plots. The plots show dark matter fraction against luminosity, effective radius, surface brightness, stellar disk length, central effective brightness, total HI mass, HI radius, and flat rotation velocity. The first table are the two highest dark matter fractions (colored blue) and the two lowest (colored green). The last table shows the correlation and determination values for each datatype.
